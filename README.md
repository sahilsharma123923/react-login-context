# React Login Form using Context API

This project demonstrates how to use React Context API for managing and sharing user authentication state across components. It includes a simple login form where user data is stored in a global context and accessed by different components without prop drilling.

## Features

* React Functional Components
* Context API
* useState Hook
* User Authentication State
* Clean and Responsive UI

## What I Learned

* Creating and using Context API
* Providing and consuming context values
* Managing global state in React
* Working with React Hooks
* Component-based architecture
